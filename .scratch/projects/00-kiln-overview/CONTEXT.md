## Key Findings
1. Kiln is the rendering engine; bullish-ssg is the orchestrator
2. Two modes: Default (vault mirror) and Custom (headless CMS)
3. Key commands: generate, serve, doctor, init, dev
4. Configuration via kiln.yaml or CLI flags
5. Requires --url for production builds (SEO features)
